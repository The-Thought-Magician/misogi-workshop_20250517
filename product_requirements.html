AI Outfit Recommendation System

Personalized outfit suggestions based on user inputs, weather data, and feedback

User Workflow

sequenceDiagram participant User participant UI as Streamlit UI participant LG as LangGraph participant Weather as Weather Service participant LLM as OpenAI LLM User->>UI: Enter personal info (height, gender, location, occasion, mood) UI->>LG: Submit inputs LG->>Weather: Request weather data Weather->>LG: Return weather information LG->>LLM: Generate outfit recommendation LLM->>LG: Return outfit suggestion LG->>UI: Display outfit recommendation User->>UI: Provide rating (1-10) alt Rating < 7 and attempts < 5 UI->>LG: Request new recommendation LG->>LLM: Generate new outfit LLM->>LG: Return new suggestion LG->>UI: Display new recommendation else Rating ≥ 7 or max attempts reached LG->>UI: Display final result end

Core Components

StateGraph Nodes

  • get_weather: Fetches weather data using Tavily Search API
  • generate_outfit: Creates outfit recommendations using OpenAI LLM
  • check_rating: Processes user feedback
  • generate_result: Produces final output

Streamlit UI Features

  • Input Fields: Height, gender, location, occasion, mood
  • Rating Mechanism: Slider (1-10)
  • Process Log: Live updates
  • API Configuration: Secure key management

Weather Integration

  • Current Version: Tavily Search API
  • Previous Versions: OpenWeatherMap, Simulated data
  • Data Points: Temperature, conditions, humidity
  • Location Support: Global cities

Key Features

Personalized Recommendations

Tailored outfit suggestions based on user's physical attributes, location, occasion, and mood

Real-time Weather Integration

Up-to-date weather data influences outfit recommendations to ensure weather-appropriate attire

Feedback Loop

User ratings trigger new recommendations for continuously improving suggestions

State Management

LangGraph tracks conversation state, input history, and recommendation attempts

Interactive UI

User-friendly Streamlit interface with input controls and real-time updates

Configurable

Support for API key configuration and customizable recommendation parameters

Technology Stack

LangGraph LangChain OpenAI Streamlit Tavily API Python dotenv Requests
